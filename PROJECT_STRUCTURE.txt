WORKFLOW TO START
=================
1. Copy .env.example вҶ’ .env
2. Add Claude API key to .env
3. Run: python -m friday.main
4. Say "Hello Friday"
5. Give a command


DEBUGGING
=========
Enable detailed logging: DEBUG_MODE=true in .env
Check logs in logs/ folder
Run test scripts in scripts/ directory


FILES YOU NEED TO CREATE
========================
These are utilities and tests referenced above:

scripts/test_audio.py
- Tests microphone and audio capture
- Shows audio levels
- Helps debug audio issues

scripts/enroll_speaker.py
- Records your voice for speaker verification
- Creates voice embeddings
- Run once to enable voice recognition

scripts/download_models.py
- Pre-downloads Whisper, Piper, and embedding models
- Optional but recommended
- Avoids long wait on first startup

tests/test_*.py
- Unit tests for each module
- Useful for development and debugging
- Run with: pytest

docs/API_REFERENCE.md
- How to extend F.R.I.D.A.Y
- Adding custom tools
- Creating new capabilities

startup_scripts/
- Windows Task Scheduler XML
- Linux systemd service file
- Optional for autostart on boot

README.md
- Project overview
- Feature highlights
- Links to documentation
